# v2.3.0 Release Notes

发布日期：2026-03-31  
对应分支：`release/v2`

`v2.3.0` 聚焦于“患者端使用体验 + 报告结构化覆盖面 + 监测/时间线视图完整度”的一轮集中提升。这个版本不是单点修补，而是把首页、档案、临床护照、病程管理、报告详情、报告管理、时间线以及 OCR 解析覆盖面一起拉齐，形成更适合演示和持续迭代的当前版本。

## 版本亮点

### 1. 报告解析覆盖面继续扩展

- 扩展 `report-manager` 中 FSHD 报告解析器的覆盖范围。
- 补齐更多实验室、肺功能、膈肌超声、心电、心超、腹部超声、MRI 等报告类型的识别与字段抽取。
- 增加更细的结构化测试样例，降低回归风险。

### 2. 患者档案与临床护照体验重做

- 重构“我的档案”页面的信息组织和视觉层次。
- 优化临床护照页面与 PDF 生成内容，使聚合视图更贴近临床展示。
- 调整后端聚合逻辑，使档案、报告与护照展示更加一致。

### 3. 病程管理、报告详情与监测面板增强

- 新增实验室/呼吸/心脏的结构化监测面板组件。
- 病程管理页加入分层筛选、时间线卡片和结构化洞察。
- 报告详情页增强结构化字段与辅助展示。
- 新增报告管理页与时间线详情页，补齐“列表 -> 详情 -> 纵向浏览”的链路。

### 4. 首页、录入、登录与问答页面整理

- 首页总览、快捷入口和 hero 区信息重组。
- 统一录入页和登录注册页交互细节。
- 优化问答页布局与页面风格，使患者端主路径更连贯。

### 5. 文档与版本材料收口

- 整理根 README、英文 README 与 docs 索引。
- 新增版本历史与 release note 文档。
- 统一当前代码版本号到 `2.3.0`。
