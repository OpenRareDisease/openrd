# v1.0.0 Release Notes

发布日期：2026-03-31  
对应基线：`master` @ `420680d`

`v1.0.0` 是对当前 `master` 基线的补标记版本。它代表项目从原型和静态设计稿，进入“可运行 monorepo + 基础 API + 初始 AI 问答集成”的第一阶段，可作为后续 `release/v2` 演进的稳定出发点。

## 版本亮点

### 1. monorepo 基础设施落地

- 完成 `apps/mobile` 与 `apps/api` 的工作区组织。
- 建立根 `package.json`、统一脚本、Prettier、Husky 与 lint-staged。
- 补齐 `.env.example`、`db` 初始化脚本和基本目录结构。

### 2. 后端基础能力可运行

- 提供基础健康检查与 PostgreSQL 连接链路。
- 提供注册 / 登录基础接口与 JWT 认证能力。
- 建立最初版患者档案相关数据文件和后端模块入口。

### 3. AI 问答第一阶段集成

- 合入首版 AI 问答能力和聊天相关数据库结构。
- 为后续知识库、报告摘要和长对话能力打下后端骨架。

### 4. 协作与设计文档建档

- 引入系统架构文档、产品需求文档与 TODO 规划。
- 建立基础 Git 工作流说明和项目 README。

## 适用范围

- 适合做第一阶段演示、基础接口开发和 monorepo 结构扩展。
- 不包含 `release/v2` 后续补齐的 OCR / 报告结构化、临床护照、长期随访与 Docker 一体化交付能力。
